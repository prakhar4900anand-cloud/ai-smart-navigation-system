# AI Smart Navigation System

## 📌 Project Overview
This project is an AI-based Smart Navigation System that finds the shortest path between two locations using the A* algorithm. It also estimates travel time based on distance.

---

## 🎯 Problem
In daily life, people face difficulty in selecting the best route. This project simulates how intelligent systems like Google Maps solve this problem.

---

## 💡 Solution
The system uses the A* search algorithm to calculate the shortest path and estimate travel time efficiently.

---

## ⚙️ Technologies Used
- Python
- A* Algorithm
- Graph Data Structure
